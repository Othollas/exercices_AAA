1. Utiliser l'algorithme pour résoudre des problèmes

Vérification de la majorité



Crée un formulaire qui permet à l’utilisateur de saisir son âge.

  • Au clic sur un bouton de validation
  • Si l’âge est supérieur ou égal à 18, un message indiquant que l’utilisateur est majeur s’affiche brièvement.
  • Si l’âge est inférieur à 18 mais supérieur à 0, un message indiquant que l’utilisateur est mineur s’affiche.

Crée un jeu où l’utilisateur doit deviner un nombre aléatoire entre 0 et 100. Il a 5 essais pour deviner le bon nombre :

  • S’il se trompe, un message indique si le nombre proposé est trop petit ou trop grand.
  • S’il trouve le bon chiffre, un message de victoire s’affiche.
  • Quand il n’a plus d’essais, un message de défaite s’affiche. Prévois un bouton pour rejouer.

Vérification de conditions



Crée une fonction qui valide si une personne peut s’inscrire à une bibliothèque selon deux critères :

  • Elle a plus de 18 ans ?
  • Elle habite à Bordeaux ?
  • Affiche un message dans la console si les deux conditions sont remplies.

  • En cliquant sur un bouton, une table de multiplication de ce nombre s’affiche de 1 à 10.
  • Ajoute un bouton pour réinitialiser l’affichage.

Tableau de multiplications personnalisées



Crée une fonction qui prend deux nombres en entrée :

  • Un multiplicateur.
  • Une longueur.
  • Elle génère une liste des multiplications du nombre jusqu’à la longueur donnée (ex: 3 x 1, 3 x 2, ..., 3 x n)

Quiz sur les tableaux



Crée un quiz en 4 questions sur les méthodes de manipulation de tableaux en JavaScript.

  • L’utilisateur doit cocher la bonne réponse parmi 4 propositions, puis valider.
  • À chaque bonne réponse, on passe à la question suivante.
  • À la fin, un message signale que le quiz est terminé.
