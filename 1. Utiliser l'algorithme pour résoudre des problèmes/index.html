1. Utiliser l'algorithme pour résoudre des problèmes

Etre mineur ou majeur



Exercice 1 : Vérification de la majorité Crée un formulaire qui permet à l’utilisateur de saisir son âge. Au clic sur un bouton de validation : Si l’âge est supérieur ou égal à 18, un message indiquant que l’utilisateur est majeur s’affiche brièvement. Si l’âge est inférieur à 18 mais supérieur à 0, un message indiquant que l’utilisateur est mineur s’affiche.

Trouve le nombre



Exercice 2 : Jeu du juste prix Crée un jeu où l’utilisateur doit deviner un nombre aléatoire entre 0 et 100. Il a 3 essais pour deviner le bon nombre : S’il se trompe, un message indique si le nombre proposé est trop petit ou trop grand. S’il trouve le bon chiffre, un message de victoire s’affiche. Quand il n’a plus d’essais, un message de défaite s’affiche. Prévois un bouton pour rejouer.

Exercice 3 : Vérification de conditions Crée une fonction qui valide si une personne peut s’inscrire à une bibliothèque selon deux critères : Elle a plus de 18 ans ? Elle habite à Bordeaux ? Affiche un message dans la console si les deux conditions sont remplies.

Crée une table de multiplication.



Exercice 4 : Table de multiplication Crée un outil où l’utilisateur saisit un nombre, puis en cliquant sur un bouton, une table de multiplication de ce nombre s’affiche de 1 à 10. Ajoute un bouton pour réinitialiser l’affichage.

Crée une pyramide.



Exercice 5 : Construction d’une pyramide Crée une pyramide de 10 étages avec un caractère choisi par l’utilisateur. La pyramide doit s’afficher dans la page avec un alignement centré visuellement.

Fonction de multiplication à longueur variable.



Exercice 6 : Tableau de multiplications personnalisées Crée une fonction qui prend deux nombres en entrée : un multiplicateur une longueur Elle génère une liste des multiplications du nombre jusqu’à la longueur donnée (ex: 3 x 1, 3 x 2, ..., 3 x n).
