Classe Personne et méthode de présentation

Crée une classe Personne avec :

  • nom, âge comme attributs
  • une méthode sePresenter() qui affiche une phrase du type : "Bonjour, je m'appelle X et j'ai Y ans.

Encapsulation et gestion de l'âge

Reprends la classe Personne. Rends l'âge protégé (ou privé si non JS) et ajoute :

  • une méthode vieillir() qui augmente l’âge
  • une méthode pour récupérer l’âge

Héritage avec classe Employé

Crée une classe Employe qui hérite de Personne :

  • Ajoute un attribut poste
  • Surcharge la méthode sePresenter() pour inclure le poste

Gestion d'une liste d'objets Employés

Crée une classe Employe qui hérite de Personne :

  • Permet d'ajouter des employés dans une liste
  • Affiche tous les employés
  • Permet de chercher un employé par nom
