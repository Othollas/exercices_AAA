Créer une liste dynamiquement

Affiche automatiquement une liste d’aliments dans une balise HTML :

  • Utilise un tableau.
  • Parcours-le avec une boucle.
  • Ajoute chaque élément au DOM.

Créer un incrementeur et un decrementeur

Crée un compteur contrôlé par deux boutons : un pour ajouter, un pour retirer :

  • Gère le compteur en JavaScript.
  • Affiche sa valeur dans la page.
  • Évite qu’il passe sous zéro.

Afficher ou cacher l'image grace au DOM

Fais un bouton qui cache ou montre une image quand on clique dessus :

  • Utilise une classe CSS pour cacher.
  • Alterne cette classe au clic.
  • Change aussi le texte du bouton.

création d'un chronometre

Crée un chronomètre avec un bouton Start/Pause et un bouton Reset :

  • Utilise un intervalle pour faire défiler le temps.
  • Gère l’état de lecture/pause.
  • Remets à zéro avec Reset.
