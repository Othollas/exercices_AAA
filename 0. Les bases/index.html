HTML pour les Héros du Clavier (Niveau : Gobelin Débutant)

Une quête épique pour apprendre à créer des pages web, une balise à la fois.

Qu’est-ce que le HTML ?

HTML (HyperText Markup Language) est le langage des pages web. C’est lui qui dit au navigateur : “Voici un titre ! Voici un paragraphe ! Et tiens, voilà une image de chat, parce qu’Internet.”

Imagine que ton navigateur est un cuisinier, et HTML, la recette. Tu donnes les instructions, il prépare la page.

À quoi ça ressemble ?

Voici une page HTML minimale :

<!DOCTYPE html>
<html>
  <head>
    <title>Ma première page</title>
  </head>
  <body>
    <h1>Bonjour, monde cruel !</h1>
    <p>Je suis une balise <strong>HTML</strong>. Peur de rien.</p>
  </body>
</html>

Oui, les chevrons `<` et `>` ne sont pas là pour faire joli. Ils ouvrent et ferment les balises, les ingrédients secrets du HTML.

Les balises les plus courantes

  • <html> : Le château. Contient toute la page.
  • <head> : Le cerveau. Des infos sur la page (invisibles).
  • <body> : Le corps. Tout ce que l’utilisateur voit.
  • <h1> à <h6> : Les titres, du plus important au moins sérieux.
  • <p> : Un paragraphe. Pour parler, beaucoup parler.
  • <strong> : Pour insister. Sérieusement. Comme ça.
  • <em> : Pour faire genre... “C’est important, mais un peu subtil.”

À toi de jouer !

Crée un fichier index.html sur ton ordinateur. Copie le code ci-dessus. Ouvre-le dans ton navigateur. Tadaaa ! Tu viens de créer une page web.

Ajoute des titres, des paragraphes, écris une blague, fais-toi plaisir. Le HTML ne mord pas. (Sauf si tu oublies de fermer une balise. Là, il boude.)
